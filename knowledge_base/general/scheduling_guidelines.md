# PawPal Scheduling Guidelines

Document ID: general-scheduling-guidelines  
Applies to: Daily pet-care schedule generation

## Scheduling Rules

- Respect the owner’s available start time and total available minutes.
- Schedule fixed-time tasks at their required times.
- Never place two tasks at overlapping times.
- Order flexible tasks by priority: high, medium, then low.
- Preserve the pet’s established routine whenever possible.
- Include every task that fits safely within the available time.
- Defer the lowest-priority flexible task when the schedule is full.
- Never shorten a task below its required duration.
- Explain why a task was scheduled, moved, deferred, or blocked.

## Source Priority

Use guidance in this order:

1. Veterinarian-provided instructions
2. Pet-specific care notes
3. Owner preferences
4. General care guidance

Higher-priority instructions override conflicting general guidance.

## Validation

Before returning a plan, confirm:

- All required information is present.
- Tasks fit within the available time.
- No tasks overlap.
- Medication tasks pass safety checks.
- Every recommendation is supported by owner input or retrieved guidance.

If validation fails, revise the plan once. If it still fails, return a safe fallback and clearly explain the unresolved problem.
